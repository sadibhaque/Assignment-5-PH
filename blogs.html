Question-1: What are the different ways to select an element in the DOM?

We can select a DOM element using getElementById("name"), getElementsByClassName("name"), getElementsByTagName("name"), querySelector("name"), and querySelectorAll("name").

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gives all HTML inside an element, innerText gives only visible text, no hidden stuff. textContent gives all text, even hidden ones.

Question-3: What is event delegation in the DOM?

Event delegation means putting one event listener on the parent instead of many children. When a children is clicked, the event goes up to the parent, then we can get which child was clicked.

Question-4: What is event bubbling in the DOM?

Event bubbling means when we click an element, the event first happens on this element, then it goes to its parent, then the parents of the parents until reaches the top.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create an element using document.createElement("tag"), add it using parent.appendChild(element) or parent.append(element), and we can remove it using element.remove() or parent.removeChild(element).
